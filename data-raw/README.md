# data-raw

This folder contains raw source files used to build the database.

These files are read-only, do not edit them manually.

## top_families.csv

I selected the 44 language top families from Glottolog that include at least 20 languages. This threshold is somewhat arbitrary, but it strikes a reasonable balance: lower values would include too many very small families, while higher values would exclude some widely spoken languages.

- Source: Glottolog 5.3 (<https://glottolog.org>)

- Accessed: 2026-04-03

- Entered manually by myself (Ema Cerezo)

- Contains: top 44 language families with subfamily and child language counts
